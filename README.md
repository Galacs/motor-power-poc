# Motor Power POC

Motor Power POC board made to test power protection circuits (OCP, UVLO, OVL, RPP...), high current step-down power supply design, rf switch connectors, external FETs H-bridge and power muxes.

## IC List

- ESP32-S3Fx (Main MCU)
- QRE1113 (reflectance sensor)
- LM74502H (Vin RPP)
- TPS56A37 (12v Vout 10A)
- TPS5430 (5v Vout 2.5A)
- LDL212 (3v3 LDO)
- CJ431 (3v shunt reference)
- TPS2121 (5v power mux)
- DRV8701 (H-bridge gate driver)
- SIS412DN-T1-GE3 (10A MOSFET)
